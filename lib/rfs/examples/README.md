# Examples

## Setup

Install dependencies in the root of the project.

```shell
npm install
# or
yarn
```

## Example setup

Each folder contains a node (`index.js`) & gulp (`gulpfile.js`) example.

## Less

### node

```shell
cd examples/less
node index.js
```

### Gulp

```shell
cd examples/less
gulp
```


## PostCSS
